iuem.sequestre Installation
-------------------------------------

To install iuem.sequestre using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``iuem.sequestre`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        iuem.sequestre
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
