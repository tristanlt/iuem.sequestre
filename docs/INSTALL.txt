collective.secretskeeper Installation
-------------------------------------

To install collective.secretskeeper using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``collective.secretskeeper`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        collective.secretskeeper
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
